Distributed systems · Vector search

HoloStream Vector Search

On a five-person team, we extended a Go streaming engine with real-time KNN search and compared remote Milvus, embedded DuckDB aggregation, and peer-to-peer gRPC designs.

My main work was rewriting the backend RPC path so heterogeneous backends could run across different workers. I also contributed the initial Milvus workflow, completion signaling, correctness tools, and experiment scripts.

  • Go
  • gRPC
  • DuckDB
  • Milvus
